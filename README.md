# Hilka Aps — Landing

Marketing website for **Hilka Aps**, an indie iOS studio.

## Project documents

The repository currently holds the design specifications for the site:

- **[DESIGN.md](DESIGN.md)** — visual design system: typography, colors, components, header/footer.
- **[STRUCTURE.md](STRUCTURE.md)** — page-by-page structure for every page.
- **[CONTENT.md](CONTENT.md)** — example copy used to build and review the prototype.
- **[HANDOFF-PROMPT.md](HANDOFF-PROMPT.md)** — brief for the design/prototyping phase.

## Key decisions

- Light, Apple-style visual system; apps-grid landing page + per-app product pages.
- Languages: **English** (default) and **Ukrainian**.
- One app at launch; no social media.

## Status

Design phase — specifications under review. Implementation stack to be selected
(candidates: Next.js / Astro / Nuxt, with internationalisation and editable content).
